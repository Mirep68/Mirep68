# 🌟 Proyectos Destacados

### 🌦️ Plataforma Inteligente de Monitoreo Meteorológico

Desarrollo de una solución integral de Internet de las Cosas (IoT) para el monitoreo de variables ambientales en tiempo real mediante una red de estaciones meteorológicas.

Este proyecto involucró el desarrollo completo del sistema, desde el diseño electrónico y la programación de dispositivos embebidos hasta la transmisión, almacenamiento, visualización y análisis de datos. La plataforma permite capturar información ambiental mediante sensores conectados a microcontroladores ESP32, transmitir los datos utilizando tecnología LoRa, almacenarlos en una base de datos y visualizarlos a través de una plataforma web para el monitoreo y la toma de decisiones.

#### Principales aportes

* Programación e integración de sensores ambientales.
* Desarrollo de software embebido para dispositivos ESP32.
* Implementación de comunicación inalámbrica mediante tecnología LoRa.
* Diseño y validación de una PCB personalizada utilizando KiCad.
* Captura, almacenamiento y gestión de datos meteorológicos.
* Desarrollo de una plataforma web para monitoreo remoto.
* Visualización de datos históricos y en tiempo real.
* Pruebas, calibración e integración de hardware y software.

#### Tecnologías utilizadas

`ESP32` `LoRa` `C++` `KiCad` `HTML` `CSS` `JavaScript` `Bases de Datos` `Git`

### 😴 NightGuard: Sistema Wearable para la Detección de Microsueño

Desarrollo de un sistema wearable inteligente orientado a la detección temprana de fatiga y episodios de microsueño en conductores mediante visión artificial e inteligencia artificial embebida.

El proyecto fue concebido como una solución tecnológica para mejorar la seguridad vial mediante el monitoreo visual en tiempo real del estado de los ojos del usuario. Utilizando una cámara integrada y un modelo de Machine Learning ejecutado directamente sobre hardware embebido, el sistema es capaz de identificar condiciones asociadas a la somnolencia y generar alertas auditivas preventivas para recuperar la atención del conductor.

El desarrollo involucró investigación del problema, recolección y etiquetado de datos, entrenamiento supervisado de modelos de inteligencia artificial, integración de hardware, diseño de un sistema portátil autónomo y validación funcional del prototipo.

#### Principales aportes

* Investigación y análisis del problema de microsueño en conductores.
* Diseño de una solución wearable basada en visión artificial.
* Captura y preparación de datos para entrenamiento supervisado.
* Desarrollo y entrenamiento de un modelo de Machine Learning utilizando Edge Impulse.
* Implementación de Inteligencia Artificial Embebida (TinyML) sobre ESP32-CAM.
* Integración de hardware para captura de imágenes y generación de alertas.
* Desarrollo de un sistema autónomo alimentado mediante batería recargable.
* Construcción e integración de un prototipo funcional basado en impresión 3D.
* Validación del sistema bajo condiciones controladas de operación.

#### Tecnologías utilizadas

`ESP32-CAM` `Edge Impulse` `TinyML` `Machine Learning` `Computer Vision` `C++` `Electrónica Embebida` `Impresión 3D` `Wearables`
